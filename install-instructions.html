Install Riigikogu Desktop on Windows

Step 1: Open the Dashboard

Click the button below to open the dashboard in your browser:

Open Dashboard

Step 2: Install the App

Look for the install icon (⊕) in your browser's address bar and click it.

Or use keyboard shortcut: Ctrl + Shift + A

Step 3: Confirm Installation

Click "Install" in the popup dialog.

Step 4: Launch the App

Find "Riigikogu Desktop" in your Windows Start Menu or on your Desktop.

Supported Browsers

  • ✅ Microsoft Edge (Recommended)
  • ✅ Google Chrome
  • ⚠️ Firefox (Manual bookmark only)

Need Help?

If you encounter issues, you can always use the dashboard directly in your browser at:
