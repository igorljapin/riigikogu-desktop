# XV Riigikogu Desktop Dashboard

Professional parliamentary analysis tool optimized for Windows desktop workstations.

## Features
- Interactive parliamentary composition visualization
- Complete MP directory (101 members)
- Coalition calculator with threshold analysis
- Optimized for large displays (1920x1080 and higher)
- Works offline after first visit
- Desktop app installation available

## Installation as Desktop App (Windows)

### Method 1: Browser Install Prompt (Easiest)

1. **Open the dashboard:**
   - Visit: https://igorljapin.github.io/riigikogu-desktop/

2. **Install as app:**
   - Look for the **install icon** (⊕ or computer/app icon) in the address bar
   - Click the install icon
   - Click "Install" in the popup
   - Or use keyboard shortcut: `Ctrl + Shift + A`

3. **Launch:**
   - Find "Riigikogu Desktop" in Windows Start Menu
   - Or use desktop shortcut (if created during install)
   - Opens in dedicated window (no browser UI)

### Method 2: Manual Install via Browser Menu

**In Google Chrome:**
1. Visit: https://igorljapin.github.io/riigikogu-desktop/
2. Click menu (⋮) in top-right corner
3. Select **"Save and share"** → **"Install page as app..."**
4. Customize app name if desired (default: "XV Riigikogu Desktop Dashboard")
5. Check **"Open as window"** (recommended)
6. Click **"Install"**

**In Microsoft Edge:**
1. Visit: https://igorljapin.github.io/riigikogu-desktop/
2. Click menu (⋯) in top-right corner
3. Select **"Apps"** → **"Install this site as an app"**
4. Customize app name if desired
5. Click **"Install"**

### Method 3: Pin to Taskbar (Quick Access)

**After installing via Method 1 or 2:**
1. Open the installed app from Start Menu
2. Right-click the app icon in taskbar
3. Select **"Pin to taskbar"**
4. App now available with one click

**Alternative - Direct Pin:**
1. Find "Riigikogu Desktop" in Start Menu
2. Right-click the app
3. Select **"Pin to taskbar"**

### Method 4: Simple Bookmark (No Installation)

**If you prefer not to install:**
1. Visit: https://igorljapin.github.io/riigikogu-desktop/
2. Press `Ctrl + D` to bookmark
3. Save to Bookmarks Bar for quick access
4. (Edge only) Right-click bookmark → Select **"Pin to Start"**

## Uninstalling

**Remove the installed app:**
1. **Windows 11:** Settings → Apps → Installed apps → Find "Riigikogu Desktop" → Click ⋯ → Uninstall
2. **Windows 10:** Settings → Apps → Apps & features → Find "Riigikogu Desktop" → Uninstall
3. **From Browser:**
   - Chrome: Click ⋮ in app window → "Uninstall Riigikogu Desktop"
   - Edge: Click ⋯ in app window → "Uninstall"

## Troubleshooting

**Install icon doesn't appear:**
- Ensure you're using Edge or Chrome (latest version)
- Try accessing the site via HTTPS
- Clear browser cache and reload
- Use Method 2 (manual install via browser menu)

**App won't open offline:**
- Visit the site at least once while online
- Wait for "Service Worker registered" confirmation
- Check Windows firewall isn't blocking the browser

**App icon looks wrong:**
- Uninstall and reinstall the app
- Clear browser cache before reinstalling
